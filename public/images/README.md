# مجلد الصور

هذا المجلد مخصص لحفظ الصور المستخدمة في التطبيق:

## أنواع الصور المدعومة:
- PNG
- JPG/JPEG
- SVG
- WebP

## الصور المقترحة:
- أيقونات المسجد
- شعارات
- رموز الطقس
- أيقونات الصلوات

## كيفية الاستخدام:
ضع الصور في هذا المجلد واستخدمها في التطبيق عبر المسار:
```
/images/اسم_الصورة.png
```

مثال:
```jsx
<img src="/images/mosque-icon.png" alt="أيقونة المسجد" />
```
